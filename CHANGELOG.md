# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-01-18

### Added

- **Document Manager**: Import, manage, and tag custom practice documents.
- **Persistence**: Save documents and performance history to `localStorage`.
- **Formatted Practice Mode**: Support for code and structured text with preserved formatting.
- **Syntax Highlighting**: Automatic highlighting for code snippets.
- **Improved UX**: Drag-and-drop file import, sorting, and filtering.
- **Performance Tracking**: Visual charts for WPM and accuracy trends.

### Changed

- Standardized on `pnpm` package manager.
- Refactored state management to use Svelte 5 reactive stores.
- Updated project structure for better maintainability.

### Fixed

- Various type safety and linting issues.
- Improved accessibility for screen readers.
